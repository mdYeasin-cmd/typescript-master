# Assignment 1

This assignment contains TypeScript practice problems and short blog explanations about TypeScript concepts.

## Files

- `solutions.ts` - Contains solutions for basic TypeScript problems, including array filtering, string reversing, union types, generics, interfaces, classes, inheritance, and array intersection.
- `blog-1.md` - Explains why `any` is considered a type safety hole, why `unknown` is safer, and how type narrowing works.
- `blog-2.md` - Explains how `Pick` and `Omit` utility types help avoid code duplication and keep TypeScript code DRY.
